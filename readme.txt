Add CSS for active category in single page

== Description ==

With this plugin, you can add easily 2 styles on your single page for highlight the active category where wp_list_categories was activated.
2 textarea are available for background and link's styles.

Informations were in table wp_options with option_name like 'singlemenuactiveAdminOptions'

== Installation ==

1. Upload single-menu-active's directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings > Single Menu Active and put your CSS into textarea. Update  and reload your single page.
